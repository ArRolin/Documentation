--- title: Includes ---

Having the same header/footer across multiple pages can be a hassle when you want to make an update to your template. That's why CloudCannon has an include feature. On any HTML page you can use the following syntax to include any text based file <!-- include another_file.html -->.

Here's an example of how you would have a static header / footer across multiple pages:

First create four files - header.html, footer.html, index.html and about.html.

In header.html put the following: 

<html>
	<head></head>
	<body>
		<div id="navigation">
			<a href="index.html"> Home </a>
			<a href="about.html"> About </a>
		</div>

		<div id="content">

And in footer.html put the following:

		</div>
	</body>
</html>

Now we can include the header and footer into our other HTML pages. Let's start with the index.html page:

<!-- include header.html -->
<h1> My Home page </h1>
<!-- include footer.html -->

and the about page:

<!-- include header.html -->
<h1> About Page </h1>
<!-- include footer.html -->

Now when you makes changes to header.html, it will update on the index and about pages.
