--- title: Collections ---

Collections are like Blogging. Each Collection you define in your _config.yml will be in the collection tab in CloudCannon

If it's a Markdown file you can editing it with our Markdown editor

If it's an HTML file and your have output set to your in your _config.yml you can use the Visual Editor with Editable Regions. If you don't have output set to true you can only use the Code Editor
