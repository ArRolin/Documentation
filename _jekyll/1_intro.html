What is Jekyll?

Jekyll is a static site generator. What that means is it takes your content written in HTML, Markdown and other markup languages, passes it through templates and outputs a completley static website

The advantages of doing this are:

  • It's simple - Jekyll is a small step up from HTML so there's not huge amount to learn.
  • Secure - You don't need to worry about the latest PHP vunribility or if your site has an security hole. Your site is just static files which makes it much harder to find security holes.
  • Fast - There's no backend processing or database to access to generate your site, so Jekyll websites are extremly fast.
  • It's powerful - Jekyll opens up a world of new possibilites on static websites. You can blog, have templates, it's hard to belive these are just static websites.

To get started on Jekyll, checkout the offical website

What does CloudCannon's Jekyll integration do?

One of the downsides of using Jekyll is there's no easy way for a non-technical users to update content themselves.

On CloudCannon you can add all the normal CloudCannon editing features to a Jekyll website. CloudCannon also allows non-technical users to blog, manage collections, front matter and data.
