--- title: Add a Website ---

The first step is to add a new site to CloudCannon by clicking on the Add a New Site button.

And type in a site name

Now it's time to add files to your website. The easiest way is to upload the files is in the browser, but you can also sync them through Cloud Storage. Click Upload your website files.

Select your website files from your computer.

Now those files have uploaded to CloudCannon, it's time to edit them. Click on index.html to update the homepage.

This will bring up your website in the visual editor. We're going to set the updatable errors. Click on the code editor:

Add a class="editable" to the HTML elements you want to be updatabled. For more information on how editables behave on different elements see the editables section

When you go back to the visual editor view there will a yellow box around the sections with the the editable class.

And that's all there is to it! If you click inside the yellow box you can make updates directly on the page.
