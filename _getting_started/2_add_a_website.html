--- title: Add a Website ---

The first step is to add a new site to CloudCannon by clicking on the Add a New Site button..

And type in a site name.

Now it's time to add files to your website. The easiest way to upload files is in the browser, but you can also sync them through Cloud Storage. Click "Upload your website files."

Select your website files from your computer.

Those files have uploaded to the site's folder in CloudCannon and can be seen in the File Browser.

Now we can begin to edit the website files. Click on index.html to update the homepage.

This will bring up your website in the Visual Editor.

We're going to set the editable areas. Click on the Code Editor. (Symbol:)

In the Code Editor add a class="editable" to the HTML elements you want to be editable. (For more information on how editables behave on different elements see the editables section.)

When you go back to the Visual Editor view there will be a now yellow box around each element with the editable class.

And that's all there is to it! If you click inside the yellow box you can make updates directly on the page.
