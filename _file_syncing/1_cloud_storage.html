--- title: Cloud Storage ---

Connecting a cloud storage provider to CloudCannon allows you to work on your website on your computer and have all changes automatically synced to CloudCannon. All files changes made on CloudCannon get synced back to your storage provider so you'll always have the latest version of the website on your computer.

Dropbox

Go to your website configuration and go to storage providers and click Connect next to Dropbox.

This will bring up a dialog asking what the main source is for the inital synchronisation. This only matters if you have existing files in CloudCannon or Dropbox, select the source of files you want to keep. Click Connect Dropbox.

This will take you to Dropbox, type in your credentials to continue.

Next you need to click allow to give CloudCannon access to a folder in your Dropbox.

Your Dropbox is now connected to CloudCannon.

Put files in the folder Apps->Cloud Cannon->Your Site Name and they will be synced to CloudCannon. Any changes made on CloudCannon will be synced back to your Dropbox.

GitHub

Go to your website configuration and go to storage providers and click connect next to GitHub.

This will redirect you to GitHub, login and authorize CloudCannon access to your GitHub Account.

You'll be redirected to CloudCannon, pick a repository to connect. If you don't have one for this website, create a new one in GitHub and refresh this page.

Select the branch you want CloudCannon to pull and push changes to.

This will bring up a dialog asking what the main source is for the inital synchronisation. This only matters if you have existing files in CloudCannon or your repository, select the source of files you want to keep. Click Connect GitHub.

GitHub is now connected. Changes you push to the Git Repository will be pulled in by CloudCannon. Any changes made on CloudCannon will automatically be committed and pushed.
